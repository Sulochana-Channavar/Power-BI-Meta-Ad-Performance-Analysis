# Power-BI-Meta-Ad-Performance-Analysis
 This dashboard empowers marketing teams to optimize landing pages, refine targeting, and reallocate budgets toward the most effective formats, audiences, and time slots.
 
# Disclaimer: This is a simulated portfolio project for educational purposes. Business context and dataset are fictionalized for learning.

#  SITUATION
Meta was running multiple ad campaigns across Facebook & Instagram, but stakeholders lacked a unified view of performance. Marketing leaders couldn’t clearly see which audiences, formats, or regions were driving ROI.

# TASK
Build an interactive Power BI dashboard to track the full funnel — from impressions to purchases — and uncover actionable insights on demographics, geography, ad formats, and time trends.

# ACTION
Every Power BI concept I applied:
🔹 Funnel KPIs — CTR, Engagement Rate, Conversion Rate, Purchase Rate
🔹 Dynamic visuals — Donut chart (gender), Bar chart (age), Map (country), Calendar heatmap (seasonality)
🔹 Weekly & hourly trends — stacked columns + area charts for activity patterns
🔹 Ad type matrix — compare Video, Story, Image, Carousel across platforms
🔹 Drill-down storytelling — awareness → engagement → purchase funnel

# RESULT
📊 CTR at 11.76% — well above industry average
📊 Engagement strong at 13.56%, but only 5.21% of clicks convert → funnel leakage
📊 Young females (18–30) = most engaged audience
📊 India & Brazil = high engagement, Germany & UK = high-value conversions
📊 Video & Story ads outperform Images/Carousels
📊 Afternoons & evenings = peak ROI

# Takeaway: The dashboard highlights strong awareness but weak purchase efficiency. Recommendations include landing page optimization, retargeting strategies, and budget reallocation toward video/story formats.
